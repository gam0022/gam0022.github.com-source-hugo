Mac OS Xでjavacやjavaで文字化けした場合の対処

2012-12-19 Java, Encoding, Mac OS X

Mac OS Xでjavacやjavaで文字化けした場合の対処ですが、 次のように~/.bash_profileにaliasを設定すれば、治ると思います。

この例ではターミナルのエンコーディングがUTF-8という 前提で設定しているので、UTF-8の部分は自分の環境に合わせて変更してください。

# ~/.bash_profile
alias javac='javac -J-Dfile.encoding=UTF-8'
alias java='java -Dfile.encoding=UTF-8'

最後にsource ~/.bash_profileしてね。

私に関して言うと、Javaを書くときはEclipseを使うことにしたので、この設定はあまり役に立ってないね！
